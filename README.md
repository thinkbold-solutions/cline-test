# cline-test

Sandbox repo for testing the PLAN → EXECUTE (Cline) → VERIFY AI workflow.

## Stack

- TypeScript
- Node.js utilities (no framework)

## Purpose

Each test adds a small, well-scoped feature via the Cline CLI workflow:
1. Write Cline brief in `~/tasks/session-state.md`
2. Run `cline-handoff` → Cline executes using local 32B model on M4 Max
3. Verify output in Claude Code
